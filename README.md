# 인내타이머

## 주요기능

- 금연 ,금주 등 날짜 카운트 기능
- MQTT 서버와 통신하여 실시간 채팅하는 기능

## 수행역할
- 총제작

## 사용한 기술
- `Swift 5`, `Xcode 13`
- `CocoaMQTT`
- CentOS7 + MosquittoMQTT + Mysql
